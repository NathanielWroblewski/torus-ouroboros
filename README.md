Running locally
---

On a mac, clone the repo, and run a server:

```
$ git clone https://github.com/NathanielWroblewski/torus-ouroboros.git
$ cd torus-ouroboros
$ open http://localhost:8000 && python -m SimpleHTTPServer
```
